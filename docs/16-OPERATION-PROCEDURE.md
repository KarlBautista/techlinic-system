# Operation Procedure

1. Open a web browser and navigate to the TechClinic System website.
2. Log in using a valid Doctor, Nurse, or Admin account.
3. Upon successful authentication, the system redirects the user to the Main Dashboard.
4. From the Dashboard, the user may:
   1. View key metrics such as total patients, visits, diagnoses, and medicine inventory.
   2. Access recent patient records and complete pending diagnoses.
   3. Navigate to other modules through the sidebar menu.
5. Navigate to the Patient Records module to:
   1. View all patient records in a sortable and paginated table.
   2. Search for a specific patient by name or student ID.
   3. Filter records by department using the dropdown filter.
   4. View the detailed medical history of a selected patient by clicking on an individual record.
   5. Review visit information, diagnosis details, prescribed medication, and treatment plans.
6. Navigate to New Patient (Nurse role) to:
   1. Enter required patient information including student ID, full name, department, date of birth, address, and contact details.
   2. Utilize the auto-fill feature for returning students by entering an existing student ID.
   3. Submit the patient record for doctor review and diagnosis completion.
7. Navigate to an incomplete patient record (Doctor role) to:
   1. Select the record requiring medical assessment from the Patient Records page.
   2. Enter diagnosis details including disease, treatment plan, and prescribed medication.
   3. Mark the record status as Complete upon successful diagnosis submission.
   4. Generate a printable medical certificate or prescription for the patient.
8. Navigate to Medicine Inventory to:
   1. View current medicine stock levels, expiry dates, and status indicators.
   2. Add new medicine information including name, quantity, dosage, and expiry date.
   3. Update existing medicine details or remove outdated entries.
   4. Search and filter medicine records for quick access.
9. Navigate to Analytics to:
   1. View graphical reports such as Most Used Medicine Stocks, Patient Record Count, Top Diagnoses, and Patient Records per Department.
   2. Select a time range (weekly, monthly, or custom) to filter chart data.
   3. Analyze trends and patterns in clinic operations for informed decision-making.
10. Navigate to Notifications to:
    1. View alerts related to diseases reaching the top 10% threshold, indicating a potential outbreak.
    2. Receive warnings for low or expiring medicine stocks requiring timely restocking.
    3. Mark notifications as read or delete them to manage the alert queue.
11. Navigate to Personnel List (Admin role) to:
    1. View the complete clinic staff directory with name, role, email, sex, and account status.
    2. Search for specific personnel by name or email.
    3. Filter staff by role using the role filter dropdown.
    4. Add new clinic staff members through the inline personnel form.
12. Navigate to Activity Log to:
    1. View a chronological record of all significant system actions.
    2. Monitor user activities such as login events, record creation, and data modifications.
    3. Track system usage for accountability and audit purposes.
13. Navigate to Settings to:
    1. View and update personal account information.
    2. Change account password securely with current password verification.
    3. Upload or update a digital signature for medical documents.
14. Log out from the system after completing all tasks by clicking the Sign Out option in the profile menu at the bottom of the sidebar.
